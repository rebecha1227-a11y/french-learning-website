## 一、整体定位

本网站是一个**个人备考工具**，使用者为：
- 中文母语者，法语几乎零基础
- 目标：通过 TCF Canada 考试，达到 C1 水平
- 每日学习时间：2 小时
- 教材：Édito A1/A2（学生用书 + 教师用书 + 练习册）

所有讲义和练习题**必须用中文写**（法语例句保留法语，但说明、解析、标题全部用简体中文）。

## 三、讲义字段规范（lecture 对象）

### 3.1 必填字段

```javascript
lecture: {
  summary: "...",          // 单元概述（2-4句，中文）
  canDo: [...],            // 学完能做什么（5-8条，中文）
  grammarSections: [...],  // 语法精讲（见 3.2）
  vocabularyGroups: [...], // 词汇积累（见 3.3）
  dictation: {...},        // 默写板块（见 3.4）⭐新增
  phonetique: "...",       // 语音重点（1段，中文说明）
  tcf: "..."               // TCF Canada 应试提示（2-3句，中文）
}
```

### 3.2 grammarSections 语法精讲

每个语法点是一个对象：

```javascript
{
  title: "语法点名称（中文）",
  explanation: "详细说明（中文，2-4句）",
  
  // 可选：变位表或对比表
  table: [
    ["列标题1", "列标题2", "列标题3"],  // 第一行是表头
    ["行数据1", "数据2", "数据3"],
    // ...
  ],
  
  examples: [
    { fr: "法语例句", zh: "中文翻译" },
    // 提供 3-5 个例句，覆盖不同人称/场景
  ],
  
  pitfall: "常见错误提示（中文，1-2句）"
}
```

**语法点数量**：每单元 2-5 个语法点，按教材顺序排列。

**表格规范**：
- 动词变位表：行 = 人称，列 = 动词（最多4个同时对比）
- 对比表（如冠词、介词）：行 = 类别，列 = 性数或条件
- 表头用中文或中法双语

### 3.3 vocabularyGroups 词汇积累

```javascript
{
  theme: "词汇组主题（中文）",
  words: [
    {
      fr: "法语单词（含词性标注，如 le/la/l'）",
      zh: "中文释义",
      example: "一个法语例句（完整句子，不要太长）",
      exampleZh: "例句中文翻译（必须填写，便于初学者理解）"
    },,
    // ...
  ]
}
```

**词汇组数量**：每单元 3-5 组，按主题分类。
**每组单词数量**：6-12 个。

**注意**：
- 名词必须标注冠词（le quartier，不只写 quartier）
- 动词标原形（habiter，不只写 habite）
- 词性括号写在冠词或词后，如 l'appartement (m.)
- 每个词条必须提供 `exampleZh`，且必须是该条 `example` 的逐句准确中文翻译，禁止模板句

### 3.4 dictation 默写板块 ⭐

这是本规范的核心新增板块。每单元设置 2-4 个默写组，涵盖：
- 本单元核心生词
- 数字/日期/星期/月份（如涉及）
- 国家/国籍词（如涉及）
- 动词变位（如涉及新动词）

```javascript
dictation: {
  intro: "本单元默写说明（中文，1句）",
  groups: [
    {
      title: "默写组标题（如：核心词汇默写、动词变位默写）",
      type: "vocab" | "conjugation" | "numbers" | "phrases",
      // type 说明：
      // vocab        = 单词默写（给中文，写法语）
      // conjugation  = 动词变位默写（给主语+动词原形，写变位形式）
      // numbers      = 数字默写（给阿拉伯数字，写法语单词）
      // phrases      = 句型默写（给中文，写完整法语句子）
      
      items: [
        {
          prompt: "提示（中文或条件描述）",
          answer: "正确答案（法语）",
          hint: "记忆提示（可选，如规则说明）"
          // hint 用于学生卡住时的提示，简短
        }
      ]
    }
  ]
}
```

**各 type 的 prompt 格式**：

| type | prompt 示例 | answer 示例 |
|------|------------|------------|
| vocab | "街区/社区" | "le quartier" |
| vocab | "大学（阴性）" | "l'université (f.)" |
| conjugation | "je + habiter" | "j'habite" |
| conjugation | "ils + aimer（否定）" | "ils n'aiment pas" |
| numbers | "17" | "dix-sept" |
| numbers | "80" | "quatre-vingts" |
| phrases | "我住在一个安静的街区。" | "J'habite dans un quartier calme." |
| phrases | "我的父亲是工程师。" | "Mon père est informaticien." |

## 四、练习题字段规范（exercises 数组）

exercises 是一个数组，每个元素是一道题组（一种题型）。

### 4.1 通用结构

```javascript
{
  type: "题型名称（中文）",
  instruction: "题目说明（中文，2-3句，说清楚要做什么）",
  items: [...]   // 题目列表
}
```

### 4.2 各题型详细规范

#### 填空题（单句）
```javascript
{
  type: "填空题",
  instruction: "用 être、avoir 或 s'appeler 的正确形式填空。（说明填写规则）",
  items: [
    {
      prompt: "Je _____ Léa et j'_____ 28 ans.",
      answer: "m'appelle / ai",
      explanation: "解析（中文，说明语法原因）"
    }
  ]
}
```
**每组 4-6 题**，覆盖不同人称和变化。

#### 语法填空（段落）
```javascript
{
  type: "语法填空",
  instruction: "阅读下面的段落，在每个空格填入最恰当的词（冠词/介词/动词变位）。",
  passage: "Bonjour ! Je (1)_____ Sanae et j'(2)_____ 23 ans...",
  items: [
    {
      blank: 1,
      answer: "m'appelle",
      explanation: "说名字用 s'appeler，第一人称：je m'appelle。"
    }
  ]
}
```
**段落长度**：8-12 个空格为宜，内容围绕本单元主题。

#### 改错题
```javascript
{
  type: "改错题",
  instruction: "找出句子中的错误并改正，说明原因。",
  items: [
    {
      prompt: "Il est italienne.",
      answer: "Il est italien.",
      explanation: "il 是阳性主语，形容词应用阳性 italien，不能用阴性 italienne。"
    }
  ]
}
```
**每组 4-6 题**，每题只有一个错误。

#### 变形/变位题
```javascript
{
  type: "国籍形容词变形",   // 或"动词变位"等
  instruction: "根据主语性别，将括号内的词变为正确形式。",
  items: [
    {
      prompt: "Elle est (japonais) _____.",
      answer: "japonaise",
      explanation: "阳性 japonais 加 e → japonaise。"
    }
  ]
}
```

#### 介词填空
```javascript
{
  type: "介词填空",
  instruction: "选择正确的介词：à / au / en / aux。",
  items: [
    {
      prompt: "Elle est née _____ Sion, _____ Suisse.",
      answer: "à / en",
      explanation: "城市前用 à；la Suisse 是阴性国名，用 en。"
    }
  ]
}
```

#### 表达练习（开放题）
```javascript
{
  type: "表达练习",
  instruction: "根据情景用法语写出完整句子或段落（字数要求）。",
  items: [
    {
      prompt: "情景描述（具体场景，如：在语言交换 App 上写自我介绍）",
      modelAnswer: "参考答案（完整法语段落）",
      keyPoints: [
        "评分要点1（中文，说明语法/词汇重点）",
        "评分要点2",
        "评分要点3"
      ]
    }
  ]
}
```
**每单元 1 道表达练习**，难度匹配当前级别。

## 五、⚠️ 图片连线题的替代方案

**问题**：原教材中大量使用"看图连线""看图匹配"题型，但网站无法显示教材图片。

**解决方案**：所有图片题必须改成以下任一**纯文字题型**：

### 方案 A：文字描述连线（用字母标记）⭐ 首选

```javascript
{
  type: "连线匹配",
  instruction: "把左列的描述和右列的词语连线（在空格写出对应字母）。",
  // ⚠️ 必须加 options 字段！渲染器会把选项显示在题目右侧方框内
  options: ["a. Salut !", "b. Bonjour !", "c. Bonsoir !", "d. Au revoir !"],
  items: [
    {
      prompt: "① 你在早上 9 点走进教室",
      answer: "b. Bonjour !",
      explanation: "Bonjour 适用于白天（日落前）。"
    },
    {
      prompt: "② 晚上 8 点遇到邻居",
      answer: "c. Bonsoir !",
      explanation: "Bonsoir 用于傍晚/晚上。"
    }
  ]
}
```

**渲染效果**：左列情景描述 + 右侧灰底选项框（自动显示），每条情景后有横线供写字母。答案默认折叠，点击展开。

**注意**：`options` 数组必须包含所有选项（含干扰项），按字母 a/b/c/d 顺序排列。

### 方案 B：情景判断（替代看图判断）
把"看图说出这是什么运动"改为"读句子判断是什么运动"：
```javascript
{
  type: "情景理解",
  instruction: "读下列描述，判断对应的活动或职业（写出法语单词）。",
  items: [
    {
      prompt: "她穿着泳衣，在游泳池里练习。她喜欢什么运动？",
      answer: "la natation",
      explanation: "la natation = 游泳。"
    }
  ]
}
```

### 方案 C：分类整理（替代"看图归类"）
```javascript
{
  type: "词汇分类",
  instruction: "把下列单词分类填入对应格子。",
  // 直接在 prompt 里给出单词列表
  items: [
    {
      prompt: "把以下词分成"家庭成员"和"职业"两类：\nla mère / l'infirmière / le frère / le coiffeur / la grand-mère / l'étudiant",
      answer: "家庭成员：la mère, le frère, la grand-mère\n职业：l'infirmière, le coiffeur, l'étudiant",
      explanation: "注意区分表示关系的名词（家人）和表示工作的名词（职业）。"
    }
  ]
}
```

**原则**：所有题目必须无图片也能独立完成。遇到原教材图片题，优先选方案 A 或 B。

## 六、默写板块设计规范（详细）

### 6.1 为什么需要默写板块

背单词和背动词变位是法语学习不可绕过的环节。默写板块专门针对：
- 本单元高频生词（含词性和冠词）
- 本单元出现的所有新动词变位
- 数字、日期、星期、月份（按单元进度）
- 本单元重要句型

### 6.2 默写板块在网站上的位置与交互方式

**位置**：默写板块显示在**练习题页面的最后**（在所有练习题之后）。

**布局**：每组默写用 flexbox 横向排列，每行显示多道（vocab/conjugation 每行 2 道，numbers 每行 3 道，phrases 每行 1 道）。提示文字和下划线底部对齐，可直接导出 PDF 后在纸上或笔记软件里填写。

**用户操作流程**：
1. 导出练习题 PDF（或直接在网页上操作）
2. 看左侧提示（中文 / 数字 / 主语+动词原形），在右侧空格写法语答案
3. 全部写完后，点右上角「📋 显示答案（批改用）」按钮
4. 展开每组答案，逐题对照批改
5. 再次点击可收起答案

### 6.3 各 type 的具体样式

#### type: "vocab"（词汇默写）
```javascript
{
  title: "核心词汇默写",
  type: "vocab",
  items: [
    { prompt: "街区/社区（阳性）", answer: "le quartier", hint: "qu- 开头" },
    { prompt: "公寓（阳性）", answer: "l'appartement (m.)", hint: "以元音开头，冠词缩写" },
    { prompt: "大学（阴性）", answer: "l'université (f.)", hint: "以元音开头，记住是阴性" },
    { prompt: "海滩", answer: "la plage", hint: "" },
    { prompt: "游泳（动词不定式）", answer: "nager", hint: "-er 结尾规则动词" }
  ]
}
```

#### type: "conjugation"（动词变位默写）
```javascript
{
  title: "动词变位默写：habiter / aimer / détester",
  type: "conjugation",
  items: [
    { prompt: "je + habiter", answer: "j'habite", hint: "元音前 je → j'" },
    { prompt: "tu + habiter", answer: "tu habites", hint: "加 -es" },
    { prompt: "il + aimer", answer: "il aime", hint: "加 -e" },
    { prompt: "nous + aimer", answer: "nous aimons", hint: "加 -ons" },
    { prompt: "vous + aimer", answer: "vous aimez", hint: "加 -ez" },
    { prompt: "ils + détester", answer: "ils détestent", hint: "加 -ent（不发音！）" },
    { prompt: "je + aimer（否定）", answer: "je n'aime pas", hint: "ne → n' 在元音前" },
    { prompt: "elle + habiter（否定）", answer: "elle n'habite pas", hint: "" }
  ]
}
```

#### type: "numbers"（数字默写）
```javascript
{
  title: "数字默写（70–100）",
  type: "numbers",
  items: [
    { prompt: "70", answer: "soixante-dix", hint: "60+10" },
    { prompt: "71", answer: "soixante et onze", hint: "60+11" },
    { prompt: "80", answer: "quatre-vingts", hint: "4×20，注意末尾 s" },
    { prompt: "81", answer: "quatre-vingt-un", hint: "4×20+1，无 s" },
    { prompt: "90", answer: "quatre-vingt-dix", hint: "4×20+10" },
    { prompt: "100", answer: "cent", hint: "" }
  ]
}
```

#### type: "phrases"（句型默写）
```javascript
{
  title: "重要句型默写",
  type: "phrases",
  items: [
    { prompt: "我住在一个安静的街区。", answer: "J'habite dans un quartier calme.", hint: "habiter + dans + 不定冠词 + 名词" },
    { prompt: "我的父亲是工程师。（职业前无冠词）", answer: "Mon père est informaticien.", hint: "être + 职业，无冠词！" },
    { prompt: "她不喜欢雪。（否定句）", answer: "Elle n'aime pas la neige.", hint: "ne → n' 在元音前" }
  ]
}
```

## 七、内容质量标准

### 7.1 讲义质量标准

✅ **必须做到**：
- 语法说明用中文写，清晰易懂，避免语言学术语
- 每个语法点必须有对应的表格（变位表、对比表等）
- 每个语法点必须有 3-5 个例句，覆盖不同人称和场景
- pitfall（常见错误）必须是中国学习者实际会犯的错误，而不是泛泛的"注意变化"
- 例句选自教材原文或高度贴近 TCF Canada 考试场景
- TCF 应试提示必须具体，说明"哪类题、考什么、注意什么"

❌ **严禁出现**：
- 只列规则不给例句
- 例句孤立，不覆盖所有重要变化
- pitfall 太泛（如"注意性数一致"——要具体说为什么中国人容易错）
- 图片连线题（必须改为文字题型）
- JS 字符串内使用中文弯引号（""）——必须改为单引号（''）或转义

### 7.2 练习题质量标准

✅ **必须做到**：
- 每单元至少 5 种题型
- 必须包含语法填空（段落题），每单元 1 篇，8-12 空
- 必须包含表达练习，1-2 道，有参考答案和评分要点
- 每道题都有答案和解析（explanation），解析说清楚语法原因
- 题目覆盖单元所有语法点
- 难度递进：填空/改错在前，语法填空和表达练习在后

❌ **严禁出现**：
- 例句只有法语、没有中文翻译（词汇卡必须有 `exampleZh`）
- `exampleZh` 使用套话模板（如“例句大意：X的常见用法”）而非逐句翻译
- 题目只有答案没有解析
- 所有题都是同一题型
- 表达练习没有参考答案
- 图片依赖题（必须改为纯文字）

### 7.3 默写板块质量标准

✅ **必须做到**：
- 每单元至少 2 个默写组，建议 3-4 个
- 动词变位默写必须包含否定形式
- 数字类必须覆盖本单元出现的所有数字
- 词汇默写的 prompt 必须用中文（让学生从中文写出法语）
- hint 简短（3-5个字），只在学生可能卡住时才有帮助

❌ **严禁出现**：
- 默写只考单词不考变位
- prompt 用法语（那只是翻译练习，不是默写）
- hint 过长或直接给出答案

## 八、JS 数据格式注意事项

这是最常见的技术坑，必须严格遵守：

### 8.1 字符串引号规则

JS 字符串用双引号包裹，字符串**内部**不能出现未转义的双引号。

**中文内容常见陷阱**：
```javascript
// ❌ 错误：中文"弯引号"在 JS 字符串里会报错
explanation: "法语有两种"你"：tu 和 vous"

// ✅ 正确：改为中文单引号
explanation: "法语有两种'你'：tu 和 vous"

// ✅ 也正确：转义
explanation: "法语有两种\"你\"：tu 和 vous"
```

**常见中文内容需要注意的场景**：
- 词汇说明里引用法语词（直接写，不加引号）
- 解析里说"某某是X性"的时候，不要用中文双引号括词
- summary 里描述单元主题时，避免用""括短语

### 8.2 单元结构的闭合

每个 unit 对象以 `},` 结尾（最后一个除外），确保数组正确闭合。常见错误：
```javascript
// ❌ 上一个 unit 的最后 ] 和下一个 unit 的 id 粘在一起
]        id: "a1-u2",

// ✅ 正确：有完整的对象闭合
            ]
          },
          {
            id: "a1-u2",
```

### 8.3 table 数据格式

表格用二维数组，第一行是表头：
```javascript
table: [
  ["表头1", "表头2", "表头3"],   // 第一行 = 表头，会加粗显示
  ["数据行1列1", "数据1", "数据2"],
  ["数据行2列1", "数据2", "数据3"],
]
```

## 九、按教材写讲义的操作流程

当拿到一本教材 PDF 后，按以下流程提取内容：

### Step 1：读目录，定位单元结构
提取：单元编号、标题、页码范围、核心交际目标（Objectifs）

### Step 2：读"语法"部分（Grammaire 页）
每个语法点提取：
- 语法点名称
- 教材里的"Échauffement（热身）"→ 帮助理解引入方式
- 教材里的"Fonctionnement（规则）"→ 提取正式规则和变位表
- 教材里的"Entraînement（练习）"→ 提取题目逻辑（但改成中文化题目）
- 教师用书里的"Corrigé（答案）"→ 确认例句答案正确性

### Step 3：读"词汇"部分（Vocabulaire 页）
提取：词汇组名称、每个词的法语+中文释义、搭配用法

### Step 4：读"交际表达"框（Pour demander/dire…）
提取：功能句型，作为 vocabularyGroups 里的"句型组"或 dictation 的 phrases 组

### Step 5：读教师用书的教学目标
确认：本单元的核心教学重点、TCF 对接点、常见错误

### Step 6：构建 dictation 板块
根据本单元所有语法点和词汇，整理：
- 哪些词是高频/核心，需要默写
- 哪些动词是新动词，需要变位默写
- 是否涉及数字/日期/星期，若有则加数字默写组

### Step 7：构建 exercises 数组
按以下顺序排列题型（由易到难）：
1. 填空题（单句，巩固最基础变化）
2. 变形题（如国籍词、职业词阴阳性）
3. 介词填空（如有介词语法点）
4. 改错题（针对 pitfall 里提到的错误类型）
5. 语法填空（段落，综合考察）
6. 连线/匹配（用纯文字替代看图）
7. 表达练习（开放题，最后压轴）
