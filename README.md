# TCF Canada 法语学习网站

## 如何打开

1. 打开 `/Users/rebecha/Desktop/法语学习网站/site`
2. 双击 `index.html`
3. 浏览器里就能看到原型页面

## 当前包含

- 今日学习任务
- 任务勾选完成
- 任务顺延到明天或自定义日期
- 基于 Édito A1 / A2 的课程主线
- 每个单元的讲义与练习题
- 练习题答案折叠查看
- 单元页导出 PDF
- 学习路线图
- 题库中心占位页
- 错题复盘占位页
- YouTube 补充学习占位页

## 当前怎么用

### 今日学习

- 在 `今日学习` 页勾选已完成任务
- 如果今天做不完，可以点 `顺延到明天`
- 也可以自己选日期，再点 `改到这一天`

### 课程主线

- 在 `课程主线` 页点击任意单元卡片
- 下方会显示该单元的讲义和练习题
- 点 `查看答案` 可以展开答案
- 点 `导出 PDF` 可以用浏览器打印保存
